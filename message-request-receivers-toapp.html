Message: request / receivers (client -> app)

Back to overview

Request the list of receivers (e.g. contacts, groups, etc.) and subscribe to updates.

This message SHALL be sent by the client as soon as it needs to access available receivers (e.g. contacts, groups, distribution lists, etc.).

This usually SHOULD be done at very beginning of the connection to display the contact list and associate available conversations with the receivers.
