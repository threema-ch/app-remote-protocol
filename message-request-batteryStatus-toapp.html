Message: request / batteryStatus (client -> app)

Back to overview

"Deprecated: Request an immediate battery status update and subscribe to updates."

This message SHALL be sent by the client if it wants to display the battery status of the app side.

Note: This is deprecated since the update/batteryStatus message will be auto-subscribed after the handshake.
