Message: response / receivers (app -> client)

Back to overview

Return the list of receivers (contacts, groups, etc.).

This message MUST be sent by the app in case the list of receivers has been requested by the client.

Acknowledgement

This message acknowledges:

The feature mask is a bitmask indicating support for certain features.

Bit values:

  • 0x01: Audio messages
  • 0x02: Group chat
  • 0x04: Ballots
  • 0x08: File transfer
